# Quantum Chemistry Suite

An interactive computational chemistry visualization platform that helps simplify quantum concepts through simulations.

## About the Project

The Quantum Chemistry Suite is a web-based application built using Streamlit. It visualizes concepts in quantum mechanics in an interactive way.

Users can explore structures, wavefunctions and probability distributions through simulations. This makes abstract concepts easier to grasp.

The Quantum Chemistry Suite was developed as part of an assignment at VIT Bhopal.

## Features

### Bohr’s Atomic Model

- Visualization of electron orbits

- Calculation of energy levels

- Atomic radius computation

- atomic number and quantum levels in the Bohr’s Atomic Model

### Quantum Foundations

- de Broglie wavelength

- Heisenberg uncertainty principle

- Schrödinger equation

### Particle in a Box (1D)

- Wavefunction visualization

- Probability density representation

- Comparison of quantum states in Particle in a Box

### Wave Dynamics

- Time evolution of wavefunctions

- Gaussian wave packet simulation

- Superposition and interference in Wave Dynamics

### 3D Orbital Lab

- Monte Carlo simulation

- Electron cloud visualization in 3D Orbital Lab

## User Interface

The application has an simple interface. Users can adjust parameters using sliders. See immediate changes in graphs and simulations.

## Setup

1.. Clone the repository from GitHub.

2. Install dependencies:

pip install -r requirements.txt

3. Run the application:

streamlit run app.py

## How to Use

- Run the application. Open the link shown in the terminal.

The link is usually http://localhost:8501.

- Select a topic using the tabs

- Adjust parameters using sliders

- Observe changes in graphs and simulations
