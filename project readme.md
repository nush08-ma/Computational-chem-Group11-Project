# Quantum Chemistry Suite

An interactive computational chemistry visualization platform that helps simplify complex quantum concepts through simulations.

---

## About the Project

The Quantum Chemistry Suite is a web-based application built using Streamlit. It focuses on visualizing key concepts in quantum mechanics in an interactive way.

Instead of only reading theory, users can explore atomic structures, wavefunctions, and probability distributions through simulations. This makes abstract concepts easier to understand.

This project was developed as part of an academic assignment at VIT Bhopal.

---

## Features

### Bohr’s Atomic Model
- Visualization of electron orbits  
- Calculation of energy levels  
- Atomic radius computation  
- Adjustable atomic number and quantum levels  

### Quantum Foundations
- de Broglie wavelength  
- Heisenberg uncertainty principle  
- Schrödinger equation  

### Particle in a Box (1D)
- Wavefunction visualization  
- Probability density representation  
- Comparison of quantum states  

### Wave Dynamics
- Time evolution of wavefunctions  
- Gaussian wave packet simulation  
- Superposition and interference  

### 3D Orbital Lab
- Monte Carlo simulation  
- Electron cloud visualization  

---

The application has a clean and simple interface. Users can adjust parameters using sliders and immediately observe changes in graphs and simulations.

## Setup

1. Download or clone the repository from GitHub

2. Install dependencies:
pip install -r requirements.txt

3. Run the application:
streamlit run app.py

---

## How to Use

- Run the application and open the link shown in the terminal (usually http://localhost:8501)  
- Select a topic using the tabs  
- Adjust parameters using sliders  
- Observe changes in graphs and simulations
